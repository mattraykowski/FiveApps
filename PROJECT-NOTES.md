# Project: Five Apps

Five Apps is a Five Parsecs and Five Leagues companion application.

## General Instructions

- Use Elixir language
- Use Phoenix framework
- Use Ash Framework
- Use Test Driven Development
- Use Tailwind CSS
- Use DaisyUI

## Coding Style

- Follow Phoenix Formatter best practices.
- For LiveView Views, prefer to put the rendered view into a .html.heex file.
- For LiveView Components, prefer to keep the whole definition including view in the .ex file.

See USAGE_RULES.md for more rules.
